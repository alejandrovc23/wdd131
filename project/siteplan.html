FitStart – Website Planning Document

WDD 131: Dynamic Web Fundamentals

Site Name

FitStart

This name reflects a simple, motivating fitness platform designed for beginners who want to start building healthy habits.

Site Purpose

The site provides an introductory fitness guide for beginners, including simple routines, fitness tips, and a goal-tracking form that saves data using localStorage.

Scenarios

  • “What simple workout can I start with as a beginner?”
  • “Where can I track my fitness goals on this website?”

Color Schema

Primary Colors:

  • #1A73E8 – Used for headers, buttons, and accents.
  • #F2F2F2 – Used for background sections.

Typography

  • Poppins – Main headings and titles.
  • Roboto – Body text and paragraphs.
